# SQL Projects

This folder contains SQL queries for consumer insights and business analysis.

## Projects

### 1. Customer Insights Analysis
**File:** `customer-insights-analysis.sql`

**Objective:** Analyze customer behavior, purchasing patterns, and segment customers based on spending and frequency.

**Key Queries:**
- Top customers by total spending
- Monthly sales trends
- Customer segmentation by purchase frequency
- Product category performance analysis

**Skills Demonstrated:**
- JOIN operations to combine customer and order data
- Aggregate functions (SUM, COUNT, AVG)
- GROUP BY for data segmentation
- CASE statements for conditional logic
- Date formatting and filtering

**Insights Generated:**
- Identified high-value customer segments
- Tracked sales performance over time
- Analyzed product category contributions to revenue
